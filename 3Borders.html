BORDER-RADIUS

En CSS permite redondear las esquinas de los elementos HTML, como cuadros o imágenes. Se especifica un valor para definir el radio de curvatura de las esquinas, pudiendo aplicarse a todas las esquinas o a cada una de forma individual. Esto se utiliza para suavizar las esquinas y mejorar la estética visual de un diseño web.
DOTTED

En CSS se utiliza para crear bordes punteados alrededor de un elemento HTML. Estos bordes consisten en una serie de puntos separados que forman una línea punteada alrededor del borde del elemento. Este estilo de borde es útil para agregar un efecto visual sutil y decorativo a los elementos en una página web, sin que el borde sea demasiado llamativo. Puedes ajustar el ancho y el color del borde, así como otros estilos de borde, para personalizar aún más la apariencia del borde punteado según tus necesidades de diseño.
DASHED

En CSS se emplea para crear bordes con un estilo de línea discontinua alrededor de un elemento HTML. Este estilo de borde consiste en una serie de segmentos de línea intercalados con espacios, creando un efecto visual de líneas cortas y segmentadas alrededor del borde del elemento. Es útil para agregar un toque de diseño interesante y distintivo a los elementos en una página web, proporcionando un aspecto más dinámico que los bordes sólidos. Al igual que con otros estilos de borde, puedes ajustar el ancho y el color del borde para adaptarlo a tus necesidades de diseño específicas.
SOLID

En CSS se utiliza para crear bordes sólidos alrededor de un elemento HTML. Este estilo de borde consiste en una línea continua y uniforme que rodea el borde del elemento, proporcionando un aspecto limpio y definido. Es comúnmente utilizado para delinear y resaltar los límites de los elementos en una página web, brindando estructura visual y claridad al diseño. Puedes personalizar el ancho y el color del borde sólido según tus preferencias de diseño para lograr el efecto deseado en tu página web.
DOUBLE

En CSS se emplea para crear bordes con un estilo de doble línea alrededor de un elemento HTML. Este estilo de borde consiste en dos líneas paralelas que rodean el borde del elemento, creando un efecto visual distintivo y más sólido que los bordes simples. Es útil para resaltar elementos importantes o para agregar un toque decorativo y elegante al diseño de una página web. Al ajustar el ancho y el color del borde doble, puedes personalizar aún más su apariencia para que se adapte perfectamente al estilo de tu sitio web.
GROOVE

En CSS se emplea para crear bordes con un efecto tridimensional que da la apariencia de un surco o ranura tallada en el borde del elemento HTML. Este estilo de borde proporciona una apariencia sólida y destacada, similar a un relieve, lo que puede añadir profundidad y realce visual a los elementos en una página web. Es útil para resaltar y enfatizar áreas específicas, como botones o cuadros de texto, agregando un toque de estilo y distinción al diseño. Al ajustar el color y el grosor del borde, puedes personalizar aún más su apariencia para que se integre de manera armoniosa en el diseño general de tu sitio web.

Este estilo de borde proporciona una apariencia sólida y destacada, similar a un relieve, lo que puede añadir profundidad y realce visual a los elementos en una página web. Es útil para resaltar y enfatizar áreas específicas, como botones o áreas de contenido importante, agregando un toque de estilo y distinción al diseño. Al ajustar el color y el grosor del borde, puedes personalizar aún más su apariencia para que se integre de manera armoniosa en el diseño general de tu sitio web.
INSET

En CSS se emplea para crear bordes con un efecto visual que da la apariencia de estar incrustados o hundidos en el elemento HTML. Este estilo de borde crea una impresión de profundidad al hacer que el borde parezca estar en la parte inferior del elemento, lo que puede añadir un aspecto tridimensional y una sensación de relieve a la interfaz. Es útil para resaltar áreas de contenido o elementos de interfaz, brindando un aspecto elegante y distintivo al diseño de una página web. Al ajustar el color y el grosor del borde, puedes personalizar aún más su apariencia para que se adapte perfectamente al estilo de tu sitio web.
OUTSET

En CSS se utiliza para crear bordes con un efecto visual que da la apariencia de sobresalir o salir hacia afuera del elemento HTML. Este estilo de borde simula una sensación de relieve al hacer que el borde parezca estar en la parte superior del elemento, lo que puede añadir profundidad y realce visual a la interfaz. Es útil para resaltar áreas específicas de contenido o elementos de interfaz, brindando un aspecto llamativo y distintivo al diseño de una página web. Al ajustar el color y el grosor del borde, puedes personalizar aún más su apariencia para que se adapte perfectamente al estilo y la estética de tu sitio web.
NONE

En CSS se utiliza para eliminar completamente el borde alrededor de un elemento HTML. Cuando se aplica este estilo de borde, el elemento no mostrará ningún borde visible en absoluto, lo que puede ser útil para crear una apariencia más minimalista o para eliminar visualmente los límites entre los elementos. Es comúnmente utilizado cuando se desea eliminar cualquier adorno de borde predeterminado que pueda estar presente en ciertos elementos, permitiendo un diseño más limpio y simplificado. Sin embargo, es importante tener en cuenta que al eliminar el borde, también se elimina cualquier indicación visual de los límites del elemento, por lo que debe usarse con precaución para evitar confusiones en el diseño.
HIDDEN

En CSS se emplea para ocultar el borde alrededor de un elemento HTML sin eliminar su espacio en el diseño. A diferencia de border-style: none;, que elimina por completo el borde y su espacio, hidden oculta visualmente el borde pero mantiene su espacio en el diseño. Este estilo de borde puede ser útil cuando deseas mantener la estructura del diseño pero no quieres que el borde sea visible. Por ejemplo, podrías usarlo para ocultar temporalmente un borde que se muestra solo en ciertas condiciones o para evitar que un borde predeterminado interfiera con el diseño visual de tu página web. Es importante tener en cuenta que aunque el borde está oculto, su espacio sigue ocupando espacio en el diseño y puede afectar al diseño de la página, por lo que debes considerar cuidadosamente su uso en tu diseño.
