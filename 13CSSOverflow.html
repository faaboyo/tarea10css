overflow: visible;: Este es el valor predeterminado. Permite que el contenido sobresalga del contenedor si es más grande que su tamaño definido. No oculta ni agrega barras de desplazamiento, lo que significa que cualquier contenido que se extienda más allá de los límites del contenedor será visible.
overflow: hidden;: Oculta cualquier contenido que se extienda más allá de los límites del contenedor. No muestra barras de desplazamiento y simplemente recorta el contenido adicional.
overflow: scroll;: Agrega barras de desplazamiento al contenedor, independientemente de si el contenido se desborda o no. Esto garantiza que el usuario siempre pueda desplazarse para ver el contenido completo, incluso si no se desborda actualmente.
overflow: auto;: Este valor automáticamente agrega barras de desplazamiento al contenedor solo cuando es necesario, es decir, cuando el contenido se desborda. Si no hay desbordamiento, no se muestran barras de desplazamiento.
Uso Obligatorio: Cada valor de overflow tiene su propósito específico y se utiliza según las necesidades de diseño y experiencia del usuario. visible es útil cuando no es necesario ocultar contenido y se desea que cualquier contenido excedente sea visible. hidden es útil cuando se desea ocultar contenido que se desborda.
visible: A menudo se usa cuando no es necesario restringir el tamaño del contenido y se quiere que cualquier exceso sea visible.
hidden: Útil cuando necesitas ocultar contenido que se desborda del contenedor.
scroll: Ideal para garantizar que el contenido sea siempre accesible mediante la adición de barras de desplazamiento, independientemente del tamaño del contenido.
auto: Perfecto para permitir que el navegador decida si se necesitan barras de desplazamiento basándose en si el contenido se desborda o no.
